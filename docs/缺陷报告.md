## 示例缺陷 1：integration 目录找不到 admin_token fixture
- 模块：自动化框架
- 严重程度：Critical
- 复现步骤：执行 `pytest integration`
- 实际结果：报错 `fixture 'admin_token' not found`
- 预期结果：联动测试可使用共享 token fixture
- 原因分析：fixture 定义在 api_auto/conftest.py，integration 是兄弟目录不可见
- 修复：将共享 fixture 上移到根 conftest.py

## 示例缺陷 2：logout 用例污染 session token
- 模块：登录认证接口
- 严重程度：Critical
- 实际结果：退出登录用例使 session admin_token 失效，后续用例 401
- 预期结果：退出测试不影响其它用例
- 修复：新增 function 级 logout_token，退出用例使用独立 token

## 示例缺陷 3：Jenkins Allure 结果被覆盖
- 模块：CI报告
- 严重程度：Major
- 实际结果：三段 pytest 执行时 `--clean-alluredir` 导致前面结果被清空
- 预期结果：最终报告包含接口、UI、联动全部结果
- 修复：移除 pytest.ini 的全局 clean，只在构建开始清理一次
