# 接口分析（RuoYi-Vue-Pro / yudao）

> 本文档记录当前测试项目实际使用的 RuoYi-Vue-Pro 管理端接口。后端根地址为 `http://localhost:48080`，管理端接口统一由 `BaseApi` 拼接 `/admin-api` 前缀。

## 通用约定

| 项 | 值 |
|----|----|
| 后端地址 | `http://localhost:48080` |
| 管理端前缀 | `/admin-api` |
| 鉴权头 | `tenant-id: 1`、`Authorization: Bearer <accessToken>` |
| 成功业务码 | `code == 0` |
| token 字段 | 登录返回 `data.accessToken` |
| 分页字段 | `pageNo` / `pageSize` |
| 分页结果 | `data.list` / `data.total` |
| 非分页列表 | 多数返回 `data` |
| 数据库 | `ruoyi-vue-pro`，表名前缀多为 `system_*` |

## 登录认证

| 方法 | 路径 | 说明 | 返回 |
|---|---|---|---|
| POST | `/admin-api/system/auth/login` | 登录 | `data.accessToken` |
| POST | `/admin-api/system/auth/logout` | 退出 | `code=0` |
| GET | `/admin-api/system/auth/get-permission-info` | 当前用户、角色、权限、菜单 | `data.user/roles/permissions/menus` |

## 字典类型 `system_dict_type`

| 方法 | 路径 | 入参 |
|---|---|---|
| GET | `/admin-api/system/dict-type/page` | `pageNo,pageSize,name?,type?,status?` |
| POST | `/admin-api/system/dict-type/create` | `name,type,status,remark?` |
| PUT | `/admin-api/system/dict-type/update` | `id,name,type,status,remark?` |
| DELETE | `/admin-api/system/dict-type/delete?id=` | Query 参数 `id` |
| GET | `/admin-api/system/dict-type/get?id=` | Query 参数 `id` |
| GET | `/admin-api/system/dict-type/list-all-simple` | 简单列表 |

## 菜单与权限

| 方法 | 路径 | 说明 |
|---|---|---|
| GET | `/admin-api/system/menu/list` | 菜单列表，返回 `data` |
| GET | `/admin-api/system/menu/list-all-simple` | 可分配菜单简单列表 |
| POST | `/admin-api/system/menu/create` | 创建菜单 |
