# 接口分析（已用若依源码核对）

> 本文档记录所有被测接口的真实路径、方法、参数、响应字段、数据库表。
> 全部已用 `E:\ruoyi` 源码核对，与代码中的 Client 实现一一对应。

## 通用约定

| 项 | 值 |
|----|----|
| 全局前缀 | `/admin-api`（controller 在 `*.controller.admin.*` 包自动加） |
| 租户请求头 | `tenant-id: 1`（连字符） |
| 鉴权头 | `Authorization: Bearer <accessToken>` |
| 统一响应 | `{ code, data, msg }`，`code==0` 成功 |
| token 字段 | 登录返回 `data.accessToken` |
| 验证码 | 默认开启，设 `yudao.captcha.enable: false` 关闭 |

## 接口清单

### 登录认证 `/system/auth`
| 方法 | 路径 | 说明 | 需token | 入参 | 返回 |
|------|------|------|:--:|------|------|
| POST | /login | 登录 | 否 | username, password, captchaVerification? | accessToken |
| POST | /logout | 退出 | 否(头里带token) | - | - |
| GET | /get-permission-info | 当前用户信息 | 是 | - | user,roles,menus |

### 字典类型 `/system/dict-type`  表 `system_dict_type`
| 方法 | 路径 | 入参 |
|------|------|------|
| POST | /create | name*, type*, status*, remark |
| PUT | /update | id, name, type, status, remark |
| DELETE | /delete?id= | id |
| GET | /page | pageNo, pageSize, name?, type? |
| GET | /get?id= | id |

### 字典数据 `/system/dict-data`  表 `system_dict_data`
| 方法 | 路径 | 入参 |
|------|------|------|
| POST | /create | sort*, label*, value*, dictType*, status* |
| DELETE | /delete?id= | id |

### 部门 `/system/dept`  表 `system_dept`（树形，用/list非/page）
| 方法 | 路径 | 入参 |
|------|------|------|
| POST | /create | name*, parentId, sort*, status*, phone?, email? |
| DELETE | /delete?id= | id |
| GET | /list | name?, status? |

### 岗位 `/system/post`  表 `system_post`
| 方法 | 路径 | 入参 |
|------|------|------|
| POST | /create | name*, code*, sort*, status, remark |
| DELETE | /delete?id= | id |
| GET | /page | pageNo, pageSize, name?, code? |

### 用户 `/system/user`  表 `system_users`
| 方法 | 路径 | 入参 | 备注 |
|------|------|------|------|
| POST | /create | username*, password*, nickname, deptId, postIds, mobile | ⚠无status/roleIds |
| PUT | /update-status | id, status | 状态改这里 |
| PUT | /update-password | id, password | 重置密码改这里 |
| DELETE | /delete?id= | id | |
| GET | /page | pageNo, pageSize, username?, mobile? | |

### 角色 `/system/role`  表 `system_role`
| 方法 | 路径 | 入参 | 备注 |
|------|------|------|------|
| POST | /create | name*, code*, sort*, status*, remark | ⚠无menuIds |
| DELETE | /delete?id= | id | |
| GET | /page | pageNo, pageSize, name? | |

### 菜单 `/system/menu`  表 `system_menu`（树形，用/list非/page）
| 方法 | 路径 | 入参 |
|------|------|------|
| POST | /create | parentId, name, type, path, sort, status |
| DELETE | /delete?id= | id |
| GET | /list | name?, status? |

### 权限分配 `/system/permission`
| 方法 | 路径 | 入参 | 说明 |
|------|------|------|------|
| GET | /list-role-menus?roleId= | roleId | 查角色菜单id集合 |
| POST | /assign-role-menu | roleId, menuIds | 给角色分菜单 |
| GET | /list-user-roles?userId= | userId | 查用户角色id集合 |
| POST | /assign-user-role | userId, roleIds | 给用户分角色 |
