# 基础能力学习清单

> 目标：支撑应届生测试岗位面试，尤其是成都 1w 左右岗位会追问的基础。

## 1. Linux
必须掌握：
- cd/ls/pwd/cp/mv/rm/mkdir
- grep/find/tail/head
- ps/top/netstat/ss/lsof
- chmod/chown
- systemctl/journalctl 基础
练习：查看后端日志、查端口占用、定位服务是否启动。

## 2. SQL
必须掌握：
- SELECT/WHERE/ORDER BY/LIMIT
- INSERT/UPDATE/DELETE
- JOIN
- COUNT/GROUP BY
- 索引基本概念
练习：查询 system_users、system_role、system_role_menu，验证权限关系。

## 3. HTTP
必须掌握：
- GET/POST/PUT/DELETE
- Header/Body/Query Param
- 200/401/403/404/500
- token 鉴权
- JSON 响应断言
练习：用 Postman/JMeter 调登录接口并携带 Authorization 调业务接口。

## 4. 网络
必须掌握：
- IP/端口/DNS
- TCP三次握手/四次挥手（理解层面）
- HTTP 与 TCP 的关系
- 浏览器访问 localhost:48080 / localhost:80 的含义
练习：排查端口不通、服务未启动、连接超时。

## 5. Python
必须掌握：
- dict/list/string
- 函数/class/self
- 文件读取/YAML
- pytest fixture/parametrize
- requests 发接口
练习：写一个登录脚本，提取 data.accessToken 后访问 /admin-api/system/auth/get-permission-info。

## 6. 自动化测试基础
必须掌握：
- 用例独立性
- 随机测试数据
- 清理策略
- 断言设计
- Page Object
- Allure报告
- Jenkins持续集成
